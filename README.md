# Project Description
This repository contains Python programming exercises completed as part of the Data Science Bootcamp practice tasks.  
The programs demonstrate basic programming concepts such as loops, conditions, functions, user input handling, and problem solving using Python.

## Implemented Levels

# Level 1 Programs
- Level1_factorial.py
  - Calculates the factorial of a given number.

- Level1_reverse_string.py
  - Reverses a string entered by the user.

# Level 2 Programs
- Level2_fibonacci.py
  - Generates Fibonacci numbers based on user input.

- Level2_largest_num.py
  - Finds the largest number from user-entered values.

### Level 3 Program
- Level3_calculator.py
  - Simple calculator performing containing:
    - Addition
    - Subtraction
    - Multiplication
    - Division
